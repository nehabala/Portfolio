BrainPoP - An Augmented Reality tool for educators

This is an augmented reality application for android devices meant to be used by teachers who want to teach young children in a more captivating manner. With this app, children can have much more interactive learning sessions. The app works with a workbook with visual markers, on top of which 3d Models will be rendered, and the child can choose the correct options with virtual buttons.
Technologies used: Unity Game engine, Vuforia AR, Free Assets.
